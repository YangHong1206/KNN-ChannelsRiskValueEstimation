数据准备

清洗数据后选择关键指标， 要将渠道分群为“高价值低风险”、“低价值高风险”等类别， 需要结合 价值指标（如销量、活跃度）和 风险指标（如逾期率、经营状态）

###价值维度（Value）有 1.销量贡献 近1年销量、历史销量、周销量、近3月预审通过量 （反映渠道的销售能力及业务规模） 2.活跃度、活跃频次、近3月预审通过进件量 （高频活跃的渠道可能更稳定且价值高） 3.地域等级、市场等级 （高等级地区通常带来更高价值）

###风险维度（Risk） 1. 逾期风险 近1年M1+逾期率、历史M1+逾期率、近1年逾期量、历史逾期量 2. 主体经营状态（如“关闭”可能代表高风险） 3. 初始生效日期、合作到期日期（计算合作时长，长期合作可能更稳定） 4. 主营品牌（某些品牌或车源可能风险更高） 5. 渠道二级科目（不同渠道属性，风险不同，比如4S店一般来说风险低于二手车店面）

  pred <- knn(
    train = channels_train,

    cl = channels_train_labels,
    k = k_values[i]
  )

}
